# 简单的设置
  set nu
  syntax on
  set showmode
  set showcmd
  set expandtab
  set softtabstop=2
  set tabstop=2

# 插件
  Vundle
  CScope
  NERDTree

# 如何注释？
  `CTRL-V` 进入 Visual Block 模式

  选中代码块，`I`
  
  插入注释符号，`//` or `#` etc.
  
  `ESC`
