第一天去上班

第一天骑车去上班

早上一路上出了很多汗

也容易迷路 看了好多次导航

但吹吹风还是很开心

大致认识了下组里的人们

配置了下账号

跟着EADP wiki上的 New Hire Setup 和 One Box Setup 搭了一整天环境

看了好久文档

心里可紧张了

每周四会有例会

自己要学的东西实在是有太多太多
